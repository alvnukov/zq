# Governance

`zq` is maintainer-led.

## Decision Model

- Day-to-day technical decisions are made by maintainers.
- Breaking changes require explicit documentation and migration notes.
- Security-sensitive changes follow [SECURITY.md](SECURITY.md).

## Project Priorities

1. Reliability and predictable behavior
2. jq compatibility where documented
3. Performance without behavioral regressions
4. Clear, testable, maintainable Rust code
