# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.4.x | yes |
| < 1.4.0 | no |

## Reporting a Vulnerability

Please report vulnerabilities privately using one of these channels:

1. GitHub Security Advisory (preferred)
2. Direct maintainer contact if advisory flow is unavailable

Please include:

- affected version
- reproduction steps or proof-of-concept
- impact assessment
- potential fix (optional)

## Response Targets

- Initial acknowledgement: within 72 hours
- Triage decision: within 7 days
- Patch/release target: as soon as feasible based on severity

Do not disclose vulnerabilities publicly before a fix is available.
